# Net Tic Tac Toe

## Authors
Xavier Le, Gabe Gordon

## Description
This project uses sockets for a networked tic-tac-toe game. First, the server must be started. Then, two clients can start and connect to the server through a socket on localhost. After two clients have connected, the game begins. When the game is over, the server sends the result to the clients and closes the connection.
